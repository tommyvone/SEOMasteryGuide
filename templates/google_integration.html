Google Tools Integration

Set up Google Analytics and Search Console

Google Analytics Setup
Step 1: Create Analytics Account
  1. Go to Google Analytics
  2. Click "Start measuring"
  3. Enter account name and configure settings
  4. Click "Create" and accept terms
Step 2: Set Up Property
  1. Enter property name: {{ project.name }}
  2. Select your timezone and currency
  3. Click "Next"
  4. Choose business category and size
Step 3: Install Tracking Code

Add this code to the <head> section of all pages:

<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
Step 4: Verify Installation
  1. Visit your website
  2. Check Analytics Real-Time reports
  3. You should see your visit
Go to Google Analytics
Google Search Console Setup
Step 1: Add Property
  1. Go to Search Console
  2. Click "Add property"
  3. Enter your domain: {{ project.domain or 'yourdomain.com' }}
Step 2: Verify Ownership

Choose one verification method:

  • HTML file: Upload file to your server
  • HTML tag: Add meta tag to homepage
  • Google Analytics: Use existing GA code
  • Domain name provider: Add DNS record
Step 3: Submit Sitemap
  1. Create XML sitemap (e.g., using Yoast SEO)
  2. Go to Sitemaps section in Search Console
  3. Enter sitemap URL: sitemap.xml
  4. Click "Submit"
Step 4: Monitor Performance

Check regularly for:

  • Indexing status
  • Search queries and rankings
  • Mobile usability issues
  • Security issues
  • Manual actions
Go to Search Console
Pro Tips

  • Set up goal tracking for conversions

  • Link Analytics with Search Console
Search Console Best Practices
  • Check for crawl errors weekly
  • Monitor Core Web Vitals
  • Review search appearance enhancements
  • Submit new pages for indexing
  • Use URL inspection tool for troubleshooting
